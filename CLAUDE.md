## 🚫 大原則：このリポジトリには一切の変更を加えない (READ-ONLY)

このリポジトリ (`neetcode-submissions-zimzax0v`) は、
[NeetCode.io](https://neetcode.io) の **GitHub Sync 機能によって自動的に同期される専用リポジトリ**です。
中身（提出解答・ディレクトリ構成・README など）は **NeetCode 側が唯一の正 (source of truth)** であり、
人手や AI による手動変更は同期の破壊・コンフリクトの原因になります。

したがって Claude は、**このリポジトリ内のいかなるファイルも作成・編集・削除・移動・リネームしてはいけません。**

### 具体的に禁止する操作

- `Data Structures & Algorithms/` 以下を含む、**既存の提出ファイル (`submission-*.py` 等) の変更・追加・削除**
- `README.md` の編集
- 新しい解答ファイルやディレクトリの作成
- 既存ファイルのフォーマット・リファクタリング・「改善」
- `git add` / `git commit` / `git push` によるこのリポジトリへの変更の反映
- 上記に相当するあらゆる書き込み操作（Write / Edit / NotebookEdit / `mv` / `rm` / `git` 経由の変更など）

> ⚠️ **例外はありません。** 「軽微な修正」「タイポ修正」「より良い解法への置き換え」であっても行わないでください。
> 唯一の例外は、この `CLAUDE.md` ファイルそのものの作成でした（それは完了済みです）。

このリポジトリでは、**閲覧・分析・説明のみ**が許可されます。

## 📝 NeetCode の問題文について

- NeetCode.io は SPA のため、問題ページを直接フェッチしても問題文は取得できません。
- 問題の内容が必要な場合は、ユーザーに問題文を貼り付けてもらうか、
  対応する LeetCode の問題名から内容を確認してください。
- いずれの場合も、**解答をこのリポジトリに書き込むことはしません**（上記の大原則を優先）。
